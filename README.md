# Freight Exchange

Freight Exchange is a CRUD microapplication written in PHP, where users can add, edit, delete, and search for transport offers – loads and vehicles. The offers contain detailed information, including a preliminary map of the route between the loading and unloading locations, using the Google Maps API. Users can also print the transport order and save it to a formatted PDF file.
